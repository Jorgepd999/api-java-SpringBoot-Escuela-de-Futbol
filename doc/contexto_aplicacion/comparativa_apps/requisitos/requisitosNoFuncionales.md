### Requisitos No Funcionales (NF)
- NF1: Respuesta del sistema < 2 segundos.
- NF2: Persistencia en base de datos.
- NF3: La aplicación soporta multilenguaje.
- NF4: Caché para consultas frecuentes.
- NF5: El sistema deberá implementar autenticación mediante correo electrónico y contraseña.
- NF6: El sistema deberá implementar autorización basada en roles.
- NF7: El acceso a los endpoints REST deberá estar protegido frente a accesos no autenticados.
- NF8: El sistema deberá registrar errores de autenticación.
