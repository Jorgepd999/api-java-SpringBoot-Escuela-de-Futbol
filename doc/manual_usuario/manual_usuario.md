# Manual de Usuario

## Gestión de jugadores

1. Acceder al listado
2. Pulsar "Nuevo"
3. Rellenar formulario
4. Guardar

---

## Edición

Seleccionar jugador → Editar → Guardar

---

## Eliminación

Seleccionar jugador → Eliminar

---

## Generar PDF

Desde listado → Exportar PDF
